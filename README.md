# Whodunit AI

Whodunit AI is an LLM-powered mystery game prototype. The player investigates a fictional murder case by interviewing suspects, examining clues, and making a final accusation.

The main goal is to explore how generative AI can create dynamic NPC dialogue while keeping the story constrained by character knowledge, secrets, clues, and game state.

## Planned Features

- One original mystery case
- LLM-powered suspect interviews
- Structured JSON data for characters, clues, locations, and story facts
- Game state tracking
- Final accusation mechanic
- Streamlit interface
- Generated visual assets for suspects and locations
- Dialogue logging and basic analysis

## Tech Stack

- Python
- Streamlit
- JSON
- LLM API
- FAISS / vector embeddings
- Pandas
- Matplotlib or Seaborn
